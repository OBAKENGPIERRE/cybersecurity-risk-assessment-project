# Cybersecurity Risk Assessment Report

## Environment Overview

The simulated organization operates a small IT infrastructure consisting of:

- Employee workstations
- File server
- Router and firewall
- Cloud storage platform
- Email system
- Public web application

These systems support daily business operations and store sensitive information.

---

## Identified Assets

| Asset | Description | Importance |
|------|-------------|-----------|
| Workstations | Employee computers used for operations | High |
| File Server | Stores company files and documents | Critical |
| Router/Firewall | Controls network traffic | Critical |
| Email System | Business communication | High |
| Cloud Storage | Remote data storage and backup | High |
| Web Application | Customer-facing platform | High |

---

## Identified Threats

Potential cybersecurity threats include:

- Phishing attacks
- Malware infections
- Unauthorized access
- Insider threats
- Data breaches
- Denial of service attacks

---

## Identified Vulnerabilities

The following vulnerabilities may exist:

- Weak password policies
- Lack of multi-factor authentication
- Outdated software
- Poor network segmentation
- Lack of employee security awareness

---

## Risk Analysis

Risk is evaluated based on likelihood and impact.

Example:

| Risk | Asset | Likelihood | Impact | Risk Level |
|-----|------|-----------|--------|-----------|
| Phishing leading to credential theft | Email System | High | High | Critical |
| Malware infection | Workstations | High | Medium | High |
| Unauthorized file access | File Server | Medium | High | High |

---

## Recommendations

The following mitigation strategies are recommended:

- Implement multi-factor authentication
- Conduct employee security awareness training
- Deploy endpoint protection solutions
- Implement strong password policies
- Conduct regular vulnerability scans
- Apply timely security patches
