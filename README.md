# Cybersecurity Risk Assessment Project

## Overview

This project demonstrates how to conduct a cybersecurity risk assessment for a simulated small business environment.

The assessment identifies critical assets, potential threats, vulnerabilities, and evaluates the risks associated with them. Mitigation strategies are proposed to reduce the organization's exposure to cybersecurity threats.

This project demonstrates practical Governance, Risk, and Compliance (GRC) skills used by cybersecurity professionals.

---

## Objectives

- Identify critical IT assets
- Identify potential threats
- Identify system vulnerabilities
- Perform risk analysis
- Create a risk register
- Recommend mitigation strategies

---

## Frameworks Referenced

- NIST Cybersecurity Framework
- ISO 27001 Risk Management Principles
